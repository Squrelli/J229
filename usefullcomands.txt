Instrukcja uruchomienia MongoDB + tworzenie bazy i kolekcji w Compass
1. Uruchomienie MongoDB w Dockerze

Zaloguj się do Docker Hub:

docker login


(Wcześniej musisz mieć zweryfikowany email na swoim koncie Docker Hub.)

Pobierz i uruchom MongoDB:

docker run -d --name mongo -p 27017:27017 mongo:6.0


Sprawdź, czy kontener działa:

docker ps


MongoDB powinno być dostępne pod:

mongodb://localhost:27017

2. Połączenie z MongoDB przez MongoDB Compass

Otwórz MongoDB Compass.

W polu Connection String wpisz:

mongodb://localhost:27017


Kliknij Connect.

3. Utworzenie bazy projekt001 i kolekcji users

W lewym panelu kliknij Create Database.

Wpisz:

Database Name: projekt001

Collection Name: users

Kliknij Create Database.

Baza projekt001 zostanie utworzona razem z kolekcją users.
